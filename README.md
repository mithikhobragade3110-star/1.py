# Welcome Program 👋

## 📌 Description
This is a simple Python program that asks the user for their name, age, hobby and city, then displays a friendly welcome message.

## 🛠️ Features
- Takes user input using `input()`
- Stores data in variables
- Displays output using `print()`
- Uses formatted strings (f-strings)

## ▶️ How to Run
1. Install Python
2. Download the file
3. Run:
   ```
   python 1.py
   ```

## 📊 Example Output
```
What is your name? Mithilesh
How old are you? 23
What is your favorite hobby? Cricket
Where do you live? Nagpur

 Welcome Mithilesh! 
You are 23 years old and love Cricket.
I live in Nagpur.
```
